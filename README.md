# CRUD Web - Sistema de Gestión de Usuarios

Aplicación web desarrollada con Java, Spring Boot, Thymeleaf y MySQL para la gestión de usuarios mediante operaciones CRUD.

## Tecnologías

- Java
- Spring Boot
- Thymeleaf
- MySQL
- HTML5
- CSS3
- Gradle

## Funcionalidades

- Inicio de sesión
- Registro de usuarios
- Consulta de usuarios
- Actualización de información
- Eliminación de registros
- Dashboard principal
- Reproductor de música integrado
